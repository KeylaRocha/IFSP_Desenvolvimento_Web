HTML

HTML é uma linguagem para estruturação e apresentação de conteúdo para a World Wide Web

HTML 5 é a mais recente evolução do padrão que define a linguagem HTML

Esta versão traz importantes mudanças quanto ao papel do HTML no mundo da Web, através de novas funcionalidades como semântica e acessibilidade

Possibilita o uso de novos recursos antes possíveis apenas com a aplicação de outras tecnologias
Muitos recursos do HTML 5 têm sido construídos com a intenção de serem executados em dispositivos móveis como smartphones e tablets

História

Entre 1991 e 1999 o HTML se desenvolveu a partir da versão 1. Em 2008, a primeira versão pública do HTML 5 foi lançada

As primeira versões do HTML foram definidas com regras sintaticas flexíveis, o que ajudou aqueles sem familiaridade com Web. Com o tempo aumentou a tendencia em tornar a sintaxe cada vez mais rigida. Apesar disso, por questões históricas(retrocompatibilidade), os navegadores ainda hoje conseguem interpretar paginas wed desenvolvidas com as primeiras versões do HTML
